Part 2

Task, find the common letters in each three row sequence. I can start off with the compartment data frame from before.

group_coding <- 
  compartment |>
  mutate(group = rep(1:(n()/3), each = 3))

I think I’ll write a n-way %in%.

multi_in <- function(x){
  out <- x[[1]]
  for(i in seq_along(x)){
    out = x[[i]][x[[i]] %in% out]
  }
  return(out)
}
multi_in(
  list(
    c("a", "b", "c", "d"), 
    c("a", "x", "y", "c"), 
    c("a", "x", "z")
  )
)
